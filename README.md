# Magic AI Practices

This repository contains practice notebooks and materials for learning AI and machine learning concepts.

## Structure

- `maf-101/` - Contains practice notebooks (notebook-1 through notebook-5)
- `references/` - Reference materials and additional resources

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Open the notebooks in Jupyter or VS Code to start practicing
